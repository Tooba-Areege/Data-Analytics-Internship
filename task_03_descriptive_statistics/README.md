## 🎯 Objective

The objective of this task is to calculate and interpret basic descriptive statistics using the cleaned Titanic dataset from Task 02.

The analysis includes calculating the count, mean, median, minimum, and maximum values for selected numerical columns. Frequency counts were also calculated for categorical variables.

---

## 📊 Dataset

The cleaned Titanic dataset from Task 02 was used for this analysis.

The following numerical columns were analyzed:

- Age
- Fare
- SibSp
- Parch

Categorical counts were calculated for:

- Gender
- Passenger Class
- Survival Status

---

## 📈 Descriptive Statistics

The following statistical measures were calculated:

- **Count:** Number of available records
- **Mean:** Average value
- **Median:** Middle value
- **Minimum:** Smallest value
- **Maximum:** Largest value

These statistics provide a basic understanding of the distribution and range of the dataset.

---

## 🔍 Interpretation

The descriptive statistics provide an overview of the cleaned Titanic dataset. The mean and median show the central tendency of numerical variables, while the minimum and maximum values show their range. Comparing the mean and median can help identify whether the data may be influenced by unusually high or low values. Frequency counts provide an overview of categorical variables such as gender, passenger class, and survival status.
